Announcing Web-Terminal

April 10, 2020

Learning to code is an extremely rewarding task, but we know it's not always easy. Working in a basic bare-bones terminal can be tedious and frustrating for programmers of any experience level, and it certainly doesn't make the learning process easier.

That's why we are thrilled to announce the launch of Web-Terminal, a "terminal on training wheels." Web-Terminal is a browser-based terminal interface with exciting new user-friendly features and tools to make coding easier for all!

Read more about Web-Terminal's key features here!


Want to help improve Web-Terminal? Our project is open source, so head over to our Contribution Page to learn how you can get involved!

Please contact us with any comments or questions you have!
