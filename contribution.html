Contribution

Have you ever been using a terminal and thought, "Wow, it would be really convenient if I could ...?"
Now, with Web-Terminal, you can create this functionality!
Our goal is to create a terminal on training wheels, one is as user friendly as possible, and with your help we can make this happen.

Don't have an idea, but still want to contribute? Below is a current list of issues and improvements which have been identified and would be great for you to take on!

How Can I Contribute?

We here at Web-Terminal are proud to have members of the community contribute to our work. Before beginning, we ask that you read our Contribution Guide.

Identified Issues/Improvements

Create a slider in the User Preferences section in the Sidebar which allows for the user to increase or decrease the font sizes of the page.

Create a button to the side of the stdout, stdin, and stderr output tabs which allows for the user to easily write the text from the open tab to a file.

Connect the stderr output to display on the stderr tab. You can use the examples from stdout and stdin for guidance.

I have an improvement idea, but I don't have much experience...

As a new coder, you are experiencing the terminal difficulties which we are aiming to eliminate! Even if you are not sure how to implement your improvement ideas, we would love to hear it. Submit an idea here! And of course, we would love for you to try your hand at contributing to any of our other identified issues.
